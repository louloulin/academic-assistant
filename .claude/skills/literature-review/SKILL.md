# Literature Review Skill

Expert in conducting comprehensive literature reviews and synthesizing academic research.

## 何时使用

当用户要求以下任务时使用此技能：
- 进行文献综述
- 分析和综合多篇学术论文
- 识别研究主题和趋势
- 发现研究空白
- 提供未来研究方向
- 整理研究领域知识

## 能力

### 文献识别与筛选
- **全面搜索**: 跨多个数据库搜索相关文献
- **筛选标准**: 应用纳入/排除标准
- **质量评估**: 评估论文的学术质量
- **相关性判断**: 评估与主题的相关程度

### 主题分析
- **关键概念识别**: 提取核心概念和术语
- **主题分组**: 按方法论/主题/时间组织
- **演进追踪**: 追踪研究的发展历程
- **趋势识别**: 识别新兴趋势和热点

### 研究空白识别
- **未研究问题**: 识别尚未涉及的问题
- **矛盾发现**: 指出研究中的矛盾结论
- **方法局限**: 发现现有方法的局限性
- **应用空白**: 识别理论到应用的差距

### 综合与写作
- **结构化组织**: 按主题/时间/方法组织
- **批判性分析**: 不仅总结，更评价
- **模式识别**: 发现跨研究的模式
- **知识整合**: 整合多个来源的知识

## 输入格式

接受以下输入：
- **topic** (required): 研究主题或问题
- **papers** (optional): 已收集的论文列表
- **options** (optional): 选项对象
  - `maxPapers`: number - 最大分析论文数（默认50）
  - `timeRange`: string - 时间范围（如"last 5 years"）
  - `organizeBy`: string - 组织方式（theme/methodology/time）
  - `identifyGaps`: boolean - 是否识别研究空白（默认true）

## 分析维度

### 按主题组织
- 技术路线
- 应用场景
- 理论框架
- 数据类型

### 按时间组织
- 早期探索（概念提出）
- 发展期（方法改进）
- 成熟期（广泛应用）
- 前沿探索（最新方向）

### 按方法论组织
- 定性研究
- 定量研究
- 混合方法
- 实验研究
- 调查研究

## 研究空白类型

### 理论空白
- 缺乏理论框架
- 理论解释不足
- 理论与实践脱节

### 方法空白
- 方法局限性
- 缺乏比较研究
- 可重复性问题

### 应用空白
- 未探索的应用场景
- 特定群体研究不足
- 跨领域应用缺失

### 数据空白
- 数据集覆盖不全
- 特定数据类型缺失
- 数据质量问题

## 评价标准

### 优秀文献综述的特征
- **全面性**: 覆盖主要研究和观点
- **系统性**: 有明确的搜索和筛选策略
- **批判性**: 不仅总结，更评价
- **组织性**: 结构清晰，逻辑连贯
- **时效性**: 包含最新研究
- **原创性**: 提出新的见解或框架

### 常见问题
- **罗列式**: 只总结不分析
- **选择性**: 只引用支持观点的研究
- **过时**: 忽略最新研究
- **狭窄**: 范围过窄，遗漏重要研究
- **缺乏批判**: 不评价研究质量

## 最佳实践

1. **明确范围**: 清晰定义综述的范围和边界
2. **系统搜索**: 使用多个数据库和搜索词
3. **筛选透明**: 说明纳入/排除标准
4. **批判思考**: 不仅是什么，更要知道为什么和怎么样
5. **识别空白**: 明确指出缺少什么
6. **提出方向**: 基于空白提出具体未来方向
