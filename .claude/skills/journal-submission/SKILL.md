# Journal Submission Skill

Expert in academic journal selection and submission strategy.

## 何时使用

当用户要求以下任务时使用此技能：
- 选择合适的期刊投稿
- 撰写投稿信(cover letter)
- 准备投稿材料
- 了解投稿流程
- 评估期刊匹配度
- 制定投稿策略

## 能力

### 期刊推荐
- **范围匹配**: 论文主题与期刊范围匹配
- **影响因子**: 考虑影响因子和声誉
- **审稿速度**: 评估期刊的审稿周期
- **接受率**: 了解期刊的接受难度
- **开放获取**: OA期刊选择

### 投稿信撰写
- **核心要素**: 研究亮点、创新性、适合性
- **个性化**: 针对具体期刊定制
- **说服力**: 突出论文价值
- **格式规范**: 符合期刊要求

### 投稿准备
- **检查清单**: 确保所有材料齐全
- **格式要求**: 符合期刊投稿指南
- **常见错误**: 避免容易犯的错误
- **时间规划**: 合理安排投稿时间

### 审稿流程
- **流程了解**: 从投稿到发表的全过程
- **状态追踪**: 理解各种审稿状态
- **回复策略**: 如何回应审稿意见
- **申诉流程**: 必要时如何申诉

## 输入格式

接受以下输入：
- **paperInfo** (required): 论文信息
  - `title`: 论文标题
  - `abstract`: 摘要
  - `keywords`: 关键词
  - `field`: 研究领域
- **options** (optional): 选项对象
  - `targetIF`: string - 目标影响因子范围
  - `preferredType`: string - 期刊类型（OA/订阅）
  - `speed`: string - 审稿速度要求（快/中/慢）
  - `includeReasons`: boolean - 是否提供推荐理由

## 期刊选择标准

### 主要考虑因素

#### 1. 主题匹配度（最重要）
- **期刊范围(Aims & Scope)**: 论文主题必须在期刊范围内
- **近期文章**: 查看期刊近期是否发表类似主题
- **目标读者**: 你的研究是否适合期刊的读者群

#### 2. 期刊质量指标
- **影响因子(Impact Factor)**: 衡量期刊影响力
- **JCR分区**: Q1(最高)到Q4(最低)
- **CiteScore**: 另一个影响力指标
- **h-index**: 期刊的h指数

#### 3. 实务考虑
- **审稿速度**: 从投稿到首次决定的时间
- **发表速度**: 接受到在线发表的时间
- **接受率**: 通常10-30%，太低需谨慎
- **版面费**: 特别是开放获取期刊

#### 4. 声誉与可见性
- **出版社声誉**: Elsevier, Springer Nature, IEEE等
- **索引情况**: SCI, SSCI, EI等
- **开放获取**: 是否OA，OA的可见性更好

### 投稿策略

#### 分层投稿策略
1. **第一梯队**: 梦想期刊（影响因子高，接受率低）
2. **第二梯队**: 现实目标（影响因子适中，匹配度高）
3. **第三梯队**: 保底选项（接受率较高）

#### 时间规划
- **同时投稿**: ❌ 不能一稿多投
- **顺序投稿**: ✅ 被拒后投下一个
- **预印本**: ✅ 可以先发arXiv等预印本

## 投稿信要素

### 标准结构

1. **称呼**
   - Dear Editor-in-Chief,
   - 或使用编辑姓名（如果知道）

2. **开头**
   - 清楚说明投稿意图
   - 提供论文标题

3. **研究亮点**
   - 2-3个关键创新点
   - 研究的重要性和意义
   - 主要发现（简短）

4. **适合性说明**
   - 为什么适合该期刊
   - 引用期刊近期相关文章
   - 对期刊读者群的兴趣

5. **声明**
   - 原创性声明
   - 未同时他投声明
   - 利益冲突声明
   - 作者贡献声明

6. **结尾**
   - 联系方式
   - 感谢

### 注意事项
- **长度**: 通常一页（300-500词）
- **语气**: 专业、礼貌、自信但不夸大
- **具体**: 避免空泛，提供具体信息
- **定制**: 每个期刊的投稿信应有所不同

- [ ] 正文(Manuscript)

- [ ] 图(Figures) - 单独文件

- [ ] 投稿信(Cover Letter)

- [ ] 字数符合要求

- [ ] 图片分辨率≥300dpi
- [ ] 所有表格和图片有编号和标题

### 声明文件
- [ ] 所有作者已阅读并同意投稿

- [ ] 研究经伦理审批（如需要）
- [ ] 资助信息已列出

## 审稿流程

### 典型流程
1. **投稿提交** (0天)
   - 在线系统提交所有材料

2. **编辑初审** (1-2周)
   - 技术检查
   - 范围匹配评估
   - 决定: 送审或拒稿

3. **同行评审** (2-6个月)
   - 2-3位审稿人
   - 匿名评审
   - 审稿意见返回

4. **编辑决定** (1-2周)
   - Accept
   - Minor Revisions
   - Major Revisions
   - Reject

5. **修改** (如需要)
   - 修改稿提交
   - 可能再次审稿

6. **最终决定**
   - Accept
   - Reject

### 审稿状态
- **Submitted to Journal**: 已提交
- **With Editor**: 编辑处理中
- **Under Review**: 审稿中
- **Required Reviews Completed**: 审稿完成，等待编辑决定
- **Decision in Process**: 编辑正在做决定

## 回复审稿意见

### 原则
1. **礼貌感谢**: 感谢审稿人的时间和建议
2. **逐条回应**: 每条意见都要回应
3. **解释修改**: 说明如何修改的
4. **理性辩护**: 如不同意，礼貌解释原因
5. **突出修改**: 在修改稿中高亮修改部分

### 回应信结构
```
Dear Editor,

感谢您和审稿人的宝贵意见。

我们已认真考虑所有建议，并进行了如下修改：

[审稿人1意见1]
- 我们的回应: ...
- 修改位置: 第X页第Y行
- 修改说明: ...

[审稿人1意见2]
...

修改摘要: 主要修改包括...

再次感谢审稿意见。
```

## 常见问题

### Q1: 被拒后怎么办?
- **分析原因**: 范围不符/质量问题/创新性不足
- **修改论文**: 根据审稿意见改进
- **选择新期刊**: 考虑更合适的期刊

### Q2: 审稿太慢怎么办?
- **耐心等待**: 审稿需要时间，通常3-6个月
- **适当询问**: 超过预期时间可礼貌询问
- **不要催促**: 频繁催促适得其反

### Q3: 如何要求加速?
- **合理理由**: 如有竞争研究
- **礼貌措辞**: 不是要求，而是请求
- **提供证据**: 说明为什么需要加速

## 最佳实践

1. **仔细阅读指南**: 投稿前仔细阅读作者指南
2. **格式完美**: 符合所有格式要求
3. **材料齐全**: 不要遗漏任何文件
4. **耐心等待**: 审稿需要时间
5. **积极回应**: 认真对待审稿意见
6. **保持专业**: 即使被拒也要保持专业态度
